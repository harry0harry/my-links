# Linktree PV - HARRY@HARRY

Projeto privado desenvolvido para agrupar e organizar links pessoais. O código foi construído inteiramente utilizando HTML, CSS e JavaScript puros (sem a necessidade de frameworks ou dependências externas), otimizado para qualquer resolução de tela.

## 🖼️ Como Alterar as Imagens

As imagens base deste projeto devem residir na pasta `assets/`. Siga os passos abaixo para atualizar o visual:

1. **Foto de Perfil:**
   - Adicione sua imagem de rosto na pasta `assets/` e renomeie-a para `perfil.png`. 
   - A aplicação de máscara circular e borda acontece automaticamente através do CSS.

2. **Imagem de Fundo:**
   - Adicione sua imagem de plano de fundo na pasta `assets/` e renomeie-a para `base.png`.
   - **Nota:** Se a imagem `base.png` não for encontrada (ou for apagada acidentalmente), o CSS ativará um sistema de *fallback* e exibirá um padrão xadrez estilizado em preto e vermelho.
